encoder for [SuperCollider Synth Definition File Format](https://doc.sccode.org/Reference/Synth-Definition-File-Format.html)

## Installation

```
npm install synthdef-json-encoder
```

## API

- `encode(data: object, [ opts: object ]): ArrayBuffer`
  - `opts.version: number`
